## Intro

In this second meetup, we will learn the most advanced concepts of mkdocs and how to deploy your site using only open source tool.
Main topics:

* Advanced concepts
* Extra components
* How to deploy a site built with MkDocs
* MkDocs vs Sphinx comparison
